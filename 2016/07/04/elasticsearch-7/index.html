《八》elasticsearch_DSL搜索

Jul 4, 2016 | Hits

example

query

The query element within the search request body

from and size

查询的开始位子和长度,from defaults to 0, and size defaults to 10

that from + size can not be more than the index.max_result_window index setting which defaults to 10,000

sort

排序,_score to sort by score, and _doc to sort by index order.The order defaults to desc when sorting on the _score, and defaults to asc when sorting on anything else.

source filtering

只查询user和message字段

指定包含和不包含的结果字段查询

other

选项 作用
min_score a _score less than the minimum specified in min_score
version Returns a version for each search hit.
explain Enables explanation for each hit on how its score was computed.
